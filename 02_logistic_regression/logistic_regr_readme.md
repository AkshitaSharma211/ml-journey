# Logistic Regression Notes

## 📘 What I Learned

* Logistic Regression is used for classification, not regression
* It predicts probability using the sigmoid function
* Output is converted into classes using a threshold (default = 0.5)
* Works well when classes are linearly separable
* Scaling features improves model performance
* GridSearchCV helps find the best hyperparameters (like C value)
* Classification Report gives precision, recall, and F1-score

---

## 📊 Results (Iris Dataset)

* Logistic Regression → High accuracy (score= 0.9393939393939394)
* Model performs well due to clear separation between classes
* Hyperparameter tuning did not significantly change performance
